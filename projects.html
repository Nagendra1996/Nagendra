Preserving Context Based Information In Cloud Image repositories

Description

The above project is done using a java Platform, this is a project which is used to preserve the information and data in the cloud computing environment and whenever user uploads a images into the cloud a key has been generated to user called repository key using that he can store the image and any other user want to access the image he has to request the user to issue a image key and by using that he can access the image and content of the image description will be in the encrypted format so that attacker attacks he cannot modify the content.
